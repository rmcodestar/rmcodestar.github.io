#### Use

**Configure global version**

```
$ jenv global oracle64-1.6.0.39
```

**Configure local version (per directory)**

```
$ jenv local oracle64-1.6.0.39
```

**Configure shell instance version**

```
$ jenv shell oracle64-1.6.0.39
```
